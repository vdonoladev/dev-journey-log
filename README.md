## :information_source: Sobre

Este repositório é o meu **portfólio de estudos em programação** — um espaço para armazenar anotações, códigos, projetos e cursos concluídos.  
A ideia é consolidar o aprendizado, manter um histórico de evolução e facilitar consultas futuras.

## :interrobang: Motivo

Criei este repositório para **organizar e documentar minha jornada como desenvolvedor**, reunindo conteúdos de diversas plataformas e tecnologias.  
É o meu “diário de bordo” no mundo da programação.
