## 📌 Visão Geral

> *"A única maneira de aprender a programar é programando."*

**Dev Journey Log** é um repositório-portfólio que documenta minha jornada de aprendizado no desenvolvimento de software. Aqui estão reunidos anotações, exercícios, projetos práticos e materiais de cursos — organizados de forma a refletir minha evolução técnica ao longo do tempo.

Este repositório serve como:

- 📖 **Diário técnico** — registro contínuo do que aprendo
- 🗂️ **Base de consulta** — referência rápida para conceitos e implementações
- 📈 **Histórico de evolução** — prova concreta do crescimento como desenvolvedor
